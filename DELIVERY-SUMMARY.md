### **Issue:** Photos won't upload
**Solution:** 
- Check browser permissions
- Try different browser (Chrome recommended)
- Ensure file size <5MB per photo

### **Issue:** Report won't print
**Solution:**
- Use Print function (Ctrl+P or Cmd+P)
- Select "Save as PDF" as destination
- Check print preview before saving

- Standalone version stores data in memory only

### **Issue:** Feature not working
**Solution:**
- Try in Chrome or Firefox (latest version)
- Clear browser cache
- Check console for errors (F12)
