# EUI-MFG-Quick-Inspect

## NSPIRE-Compliant Rental Unit Inspection System

Federal compliance inspection tool for HUD, HOME, LIHTC, and USDA Rural Development programs.

### Quick Start

1. **Upload to GitHub**:
   - Create repository: `EUI-MFG-Quick-Inspect`
   - Upload: `index.html` and `rental-inspection-app-complete.jsx`
   
2. **Enable GitHub Pages**:
   - Settings → Pages → Source: main → Folder: / (root) → Save
   
3. **Access**:
   - https://ecmcnelley.github.io/EUI-MFG-Quick-Inspect/

### Features

- ✅ Property dropdown: P56100 + Custom
- ✅ Address dropdown: 502 & 554 South 5th Street + Custom  
- ✅ Inspector dropdown: Ellie McNelley, Julian Rubio + Custom
- ✅ Room-by-room NSPIRE inspections
- ✅ Photo upload with comments (3 per row in reports)
- ✅ Work order tracking
- ✅ PDF report generation with logo
- ✅ Federal compliance checking

### Browser Requirements

- Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+
- Internet connection for initial load
- JavaScript enabled
